# ascii_cam_app

This is a camera application using terminal to show the frame(Ascii camera).
## Start
- build the project
```
$ mkdir build
$ cmake ..
$ make
```

- start the camera
$ ./ascii_cpp_cam

- quit the camera
press 'q'
